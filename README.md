# Weltschmerz
Weltschmerz is a world generator for Terra.

* weltschmerz-core: the core
* weltschmerz-noise: noise generator using OpenSimplex/Joise
* heightmap-generator: for generating heightmaps.

Dependency tree:
* weltschmerz-core
  * heightmap-generator
    * weltschmerz-noise
      * SudoPlayGames/Joise
